# Backend
Nest.js 프레임워크에서 동작하는 백엔드 애플리케이션입니다.<br>
`../api-docs` 에서 Swagger 문서를 참고해주세요.

## Environmental variables
```
MONGO_URI = "MongoDB URI"
MONGO_USERNAME = "MongoDB Username"
MONGO_PASSWORD = "MongoDB Password"
JWT_SECRET = "JWT Secret Key"
JWT_ACCESS_EXP = '30m'
JWT_REFRESH_EXP = '7d'
LOCATION_RANGE = 100
```
